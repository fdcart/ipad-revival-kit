# iPad Revival Kit

An open-source toolkit for giving old iPads a second life.

This project does **not** install unsupported versions of iPadOS. Instead, it turns old iPads into useful cultural, educational and practical devices through lightweight web tools, kiosk workflows and reuse modes.

> When Apple ends support, culture begins repair.

## Modules

- Revival Video Looper
- iPad Diagnosis
- Museum Mode
- Smart Home Mode
- Reading & Writing Mode
- Hacker Mode
- Kids / School Mode
- Digital Art Frame

## How to use

1. Open `index.html` in Safari.
2. Open a module.
3. Use Safari → Share → Add to Home Screen.
4. Use Guided Access for kiosk mode.
