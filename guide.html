iPad Revival Guide

A practical guide for turning old iPads into useful cultural devices.

Diagnosis

Check model, maximum iPadOS, battery, Wi-Fi, Bluetooth, Safari compatibility and storage. Weak battery devices are best used plugged in as kiosks.

Museum Mode

Use old iPads as artwork labels, video art screens, audio guides, archive stations, QR code screens and visitor feedback points.

Smart Home Mode

Use the iPad as a wall dashboard for Home Assistant, lights, cameras, music, calendar and weather.

Reading & Writing

Use it as a PDF reader, recipe display, e-book machine, quiet notebook or Bluetooth keyboard typewriter.

Hacker Mode

Use the iPad as a thin client for SSH, remote desktop, server dashboards, Raspberry Pi control and iSH experiments.

Kids / School Mode

Create a controlled learning tablet with offline videos, reading material, drawing tools, calculator, dictionary and Guided Access.

Digital Art Frame

Use the iPad as a permanent art display for still images, video loops, generative art pages, portfolio screens and studio moodboards.

Kiosk Setup

  1. Open the desired page in Safari.
  2. Tap Share → Add to Home Screen.
  3. Open the installed icon.
  4. Enable Guided Access in Accessibility settings.
  5. Triple-click the Home or Side button to lock the iPad into the app.
